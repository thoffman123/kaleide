# kaleide.app

Landing page and privacy policy for [Kaleide](https://kaleide.app) — a guided meditation app for iPhone and iPad.

## Auto-deploy

This repo is connected to Porkbun static hosting. Every push to `main` deploys automatically to **https://kaleide.app** — no manual upload or build step needed.

| File | URL |
|---|---|
| `index.html` | https://kaleide.app |
| `privacy.html` | https://kaleide.app/privacy.html |

## Making changes

Edit the files locally, commit, and push to `main`:

```bash
git add -p
git commit -m "your message"
git push
```

The site updates within a minute or two of the push.
